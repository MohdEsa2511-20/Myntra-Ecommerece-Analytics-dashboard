About This Project

E-Commerce Analytics Dashboard

This project is a Business Intelligence Dashboard developed using Python, Flask, MySQL, SQLAlchemy, Pandas, Plotly, and Bootstrap 5.

The dashboard analyzes e-commerce sales data and provides interactive visualizations to help businesses understand revenue, customers, products, and overall sales performance.

Key Features

  • Interactive Dashboard
  • Sales Analytics
  • Customer Insights
  • Product Performance
  • Revenue Analysis
  • Payment Method Analysis
  • Monthly Sales Trend
  • Responsive Bootstrap Design

Technology Stack

Frontend HTML5, CSS3, Bootstrap 5
Backend Python, Flask
Database MySQL
Libraries Pandas, SQLAlchemy, Plotly
